# Azure DevOps & AI Learning Platform

[![Live Demo](https://img.shields.io/badge/Live_Demo-Open_in_GitHub_Pages-blue?style=for-the-badge&logo=github)](https://vikram512700.github.io/azure-devops-pro/)

This is built like a real engineer would build a learning product for other engineers.
Every chapter tries to answer one practical question:

`Can you explain it, build it, troubleshoot it, and defend it in an interview?`

## What This Repo Feels Like
- A portfolio-grade DevOps academy, not just a notes dump.
- A study path shaped around daily work in GCC and enterprise platform teams.
- A place where every topic is tied back to delivery, incidents, migration work, or hiring signals.

## What You Will Find
- Real-world Linux, Git, Azure, Terraform, CI/CD, monitoring, and DevSecOps notes.
- End-to-end projects that show how an engineer actually finishes a system.
- Docs that use AWS-to-Azure comparisons, production scenarios, and interview-ready framing.
- Job-market driven chapters that connect learning to Hyderabad hiring patterns.

## Why It Exists
- To help you move from “I know the tool” to “I can ship the solution.”
- To turn theory into portfolio evidence.
- To make interview prep feel like production practice.

## Local Run
```bash
cd frontend
npm install
npm run dev
```

Open:
```text
http://localhost:3000
```

## AI Features
- The AI Interviewer and JD Analyzer use a user-provided Gemini API key.
- Add it from `Dashboard -> Settings` inside the app.
- The key stays in browser storage for local use.

## Docs Tone
- Every chapter should read like a senior engineer explaining how a real system was built.
- Prefer “what happened in production” over abstract textbook wording.
- Use short code blocks, practical commands, and clear finish criteria.
