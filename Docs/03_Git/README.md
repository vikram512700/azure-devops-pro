## 📌 Quick Reference Card

```bash
# Daily DevOps Git Cycle
git status                    # check state
git pull --rebase origin main # sync with remote
git checkout -b feature/xyz   # new branch
git add -p                    # stage hunks selectively
git commit -m "feat: message" # commit with convention
git push -u origin feature/xyz # push and track
```
