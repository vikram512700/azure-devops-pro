## 📚 What's Covered

This guide includes 12 interactive sections:

1. **📊 Overview** - Complete picture of AI/ML in modern software development
2. **🤖 AI & ML Basics** - Understanding AI, machine learning types, and algorithms
3. **📈 Data Science** - Data workflows, preprocessing, and feature engineering
4. **🏋️ Model Training** - Training process, hyperparameters, and infrastructure
5. **💬 LLMs** - Large Language Models, APIs, and practical use cases
6. **🔍 RAG** - Retrieval-Augmented Generation and vector databases
7. **🔌 MCP** - Model Context Protocol for AI tool integration
8. **⚖️ Model Weights** - Parameters, file formats, and quantization
9. **⚡ Inference** - Serving predictions in batch and real-time
10. **🔄 MLOps** - ML + DevOps lifecycle, monitoring, and drift detection
11. **🛠️ DevOps Integration** - Complete Azure stack and CI/CD pipelines
12. **📦 Project Example** - End-to-end fraud detection system implementation
