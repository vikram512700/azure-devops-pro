# ⚡ Quick Start - Get Live in 5 Minutes

## 🎯 Fastest Way to Deploy

### Option 1: GitHub Web Interface (No Git Required) ⭐ RECOMMENDED

**Time: 3 minutes**

1. **Create Repository**
   - Go to https://github.com/new
   - Name: `ai-ml-devops-guide`
   - ✅ Public
   - ✅ Add README
   - Click "Create repository"

2. **Upload File**
   - Click "Add file" → "Upload files"
   - Drag and drop `index.html` from the downloads
   - Commit: "Add guide"

3. **Enable Pages**
   - Settings → Pages
   - Source: `main` branch, `/ (root)`
   - Save

4. **Done! 🎉**
   - Live at: `https://YOUR-USERNAME.github.io/ai-ml-devops-guide/`
   - Wait 2 minutes for first deployment

---

### Option 2: Git Command Line (For Developers)

**Time: 5 minutes**

```bash
# 1. Clone or download the files to a folder
cd ai-ml-devops-guide

# 2. Run the deploy script
./deploy.sh

# 3. Follow the prompts
# The script will guide you through everything!
```

---

### Option 3: GitHub Desktop (Visual Git Client)

**Time: 4 minutes**

1. Download [GitHub Desktop](https://desktop.github.com/)
2. File → New Repository
   - Name: `ai-ml-devops-guide`
   - Initialize: Yes
3. Copy `index.html` into the folder
4. Commit: "Add guide"
5. Publish to GitHub (make it public)
6. Go to GitHub.com → Your repo → Settings → Pages
7. Enable Pages: `main` branch

## 🆘 Quick Troubleshooting

**404 Error?**
- Check file is named `index.html`
- Verify Pages is enabled
- Wait 3 minutes

**CSS Broken?**
- Clear cache (Ctrl+Shift+R)
- Try incognito mode
- Re-upload `index.html`

**Still Not Working?**
- Check GitHub Actions tab for build errors
- See detailed DEPLOYMENT.md guide
