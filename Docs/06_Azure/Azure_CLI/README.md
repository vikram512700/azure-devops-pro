# Azure CLI Operations Guide

This folder is for the person working from a terminal, not a portal tour.
It shows how Azure behaves when you are provisioning, validating, troubleshooting, and automating real infrastructure.

## What This Folder Covers
- CLI setup and authentication
- Compute, networking, storage, and databases
- Containers, AKS, and App Service
- Monitoring, security, and advanced scripts

## Real-World Lens
- Use Azure CLI when speed and repeatability matter.
- Treat every command as something you might later place into a script or pipeline.
- Verify the resource state after every change.

## Why It Matters
- The CLI is what turns cloud work into reliable automation.
- It is the difference between knowing Azure and operating Azure.
